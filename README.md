# CareCode Open Hospital Management Information System

## Overview

A comprehensive Hospital Information Management System serving over 40 healthcare institutions since 2004. This open-source solution provides modules for all aspects of hospital management, from patient care to administrative functions.

## Key Features

* **Electronic Medical Records (EMR)**
* **Pharmacy & Laboratory Information Management**
* **Inpatient & Outpatient Management**
* **Appointment & Theatre Management**
* **Human Resource & Payroll Management**
* **Inventory & Asset Management**

## Quick Start

### Installation
1. Clone the repository
2. Configure a blank database connection in your application server
3. Run the project - it will guide you through initial setup

### System Requirements
- Java Enterprise Edition compatible server
- MySQL or MariaDB database
- Modern web browser

## Current Version
Version: 3.0.0.20240813.1

## Project History

Started in 2004 by Dr. M H B Ariyaratne as an EMR for general practice. Evolved through community feedback and became open source in 2012 under MIT license. Now serves diverse healthcare institutions from small clinics to major hospitals.
