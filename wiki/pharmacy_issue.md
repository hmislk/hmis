# pharmacy_issue
Source: `/pharmacy/pharmacy_issue.xhtml`
## Configuration Options
- `Add the Institution Code to the Bill Number Generator` - controls `add the institution code to the bill number generator` behaviour.
- `Bill Number Generation Strategy for Disposal Issue - Separate Bill Numbers for Issuing Department` - controls `bill number generation strategy for disposal issue - separate bill numbers for issuing department` behaviour.
- `Bill Number Generation Strategy for Disposal Issue - Separate Bill Numbers for Logged Department` - controls `bill number generation strategy for disposal issue - separate bill numbers for logged department` behaviour.
- `Bill Number Generation Strategy for Disposal Issue - Separate Bill Numbers for Logged and Issuing Department Combination` - controls `bill number generation strategy for disposal issue - separate bill numbers for logged and issuing department combination` behaviour.
- `Consumption - Show Rate and Value` - controls `consumption - show rate and value` behaviour.
- `Enable Drawer Manegment` - controls `enable drawer manegment` behaviour.
- `Enable the Special Privilege of Canceling CC Bills` - controls `enable the special privilege of canceling cc bills` behaviour.
- `Enable the Special Privilege of Canceling OPD Bills` - controls `enable the special privilege of canceling opd bills` behaviour.
- `Immediate Refund Request for OPO Bills of Any Status` - controls `immediate refund request for opo bills of any status` behaviour.
- `Manage Costing` - controls `manage costing` behaviour.
- `Pharmacy Disposal is by Cost Rate` - controls `pharmacy disposal is by cost rate` behaviour.
- `Pharmacy Disposal is by Purchase Rate` - controls `pharmacy disposal is by purchase rate` behaviour.
- `Pharmacy Disposal is by Retail Rate` - controls `pharmacy disposal is by retail rate` behaviour.
- `Set the Original Bill PaymentMethod to Cancelation Bill` - controls `set the original bill paymentmethod to cancelation bill` behaviour.
- `Set the Original Bill PaymentMethod to Refunded Bill` - controls `set the original bill paymentmethod to refunded bill` behaviour.
## Privileges
- `ChangeReceiptPrintingPaperTypes` - requires `ChangeReceiptPrintingPaperTypes` permission.
- `ConsumptionViewRates` - requires `ConsumptionViewRates` permission.
