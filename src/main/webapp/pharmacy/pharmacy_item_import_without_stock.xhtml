General Instructions

Please ensure your Excel file follows the format below. Leave the first row for column headings.

Use only *.xlsx files. Do not use *.xls files.
Excel Column Field Instructions
A Serial No Provide a unique serial number for each item.
B Category Enter the category. If not found, it will be created automatically.
C Product Enter the full product name.
D Code Enter the system or catalogue code for the product.
E Bar Code Enter the barcode if available. Leave blank if not applicable.
F Generic Name Recommended to fill. Can be left blank if not known.
G Strength Enter a numeric value (e.g., 500, 1, 0.5). Must not be blank. Decimal values are allowed.
H Strength Unit Enter the unit of strength (e.g., mg, ml, g, %, IU, #, /Unit). Must not be blank. Use standard abbreviations.
I Pack Size Enter a numeric value (e.g., 1, 10, 20). Do not enter text like “Pack”.
J Issue Unit Enter the unit in which the item is issued (e.g., Tablet, Bottle, Unit).
K Pack Unit Enter the unit for the pack (e.g., Pack, Box, Strip).
L Distributor Enter the distributor name if available.
M Manufacturer Enter the manufacturer name if available. Can be left blank.
N Importer Enter the importer name if available. Can be left blank.
O Department Type (Optional) Must be one of the following: Clinical, NonClinical, Pharmacy, Lab, Channelling, Opd, Inward, Theatre, Etu, CollectingCentre, Ict, Other.
